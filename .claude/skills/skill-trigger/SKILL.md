name: skill-trigger
description: スキル記録
---

会話でスキルの記録をリクエストされたときの処理を定めています。
複数のスクリーンショットがある場合は、そのそれぞれについて下記の処理を実施してください。

**重要**: 処理の途中でユーザーへのテキスト出力を一切行わないでください。画像の読み取り結果、日付の決定、コマンド実行の報告などの途中経過は出力せず、すべての処理が完了した後に「出力例」の形式のみで結果を出力してください。

## 1. 日付の決定

- プロンプトで日付を指定されたときには、その日付を利用してください。
   - 日付は曜日や昨日・今日・明日などの表現で指定されることもあります。
- 未指定のときは以下の手順で決定してください。
   1. まず、スクリーンショットから時刻を読み取る（例: 23:11）。
   2. 当日の日付とスクリーンショットの時刻を組み合わせた日時が、現在日時よりも未来かどうかを判定する。
      - 例: 現在が 2/17 0:37 で、スクリーンショットの時刻が 23:11 の場合、2/17 23:11 は未来になる。
   3. 未来になる場合は、前日の日付を採用する（上記の例では 2/16）。未来でなければ当日の日付を採用する。
- 日付はすべてYYYY-MM-DDの形式に変換してください。

## 2. 時刻の決定

スクリーンショットの左上から、スキル発生の時刻を hh:mm 形式で読み取ってください。

## 3. ポケモン名とスキル回数の決定

スクリーンショットの下部の白い四角で囲われた範囲から、ポケモン名を読み取ってください。たとえば、「おてぼんね」がスキル発動した場合は、「おてぼんねのメインスキル！」と出力されます。「のメインスキル」の手前の部分を採用してください。

スキル回数は白い四角の個数で判定してください。場合によっては、片方にテキストが表示されていない可能性があるためです。スキル回数は多くても2回です。

## 4. スクリプトの実行

以下のコマンドを実行してスプレッドシートに記録してください。

```
venv\Scripts\python skill_recorder.py YYYY-MM-DD hh:mm ポケモン名 回数
```

例: `venv\Scripts\python skill_recorder.py 2026-02-01 14:30 おてぼんね 2`

   - コマンドの出力結果をユーザーに伝えてください
   - コマンドに失敗したときは出力結果とともに、そのことをユーザーに伝えてください

読み取れなかった場合は、読み取れなかった項目をユーザーに伝えてください。

### 出力例

```
📝記録しました
* 2月1日 14:30 ‼️おてぼんね
* 2月1日 14:30 ❗速枕
```

※2回のときは‼️、1回のときは❗をポケモン名の前に付与してください。
