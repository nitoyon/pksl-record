---

会話でエナジー読み取りをリクエストされたときに、次の処理をしてください。

## 1. 日付の決定

会話では、日付が指定されます。日付は曜日や昨日・今日・明日などの表現で指定されることもありますが、これらはすべてYYYY-MM-DDの形式に変換してください。

## 2. 添付された画像の読み取り

スクリーンショット画像から、以下の4つの値を読み取ってください。
複数のスクリーンショットが渡された場合は、それぞれに対して読み取ってください。

- **朝昼晩の区別**:
   - 画面の左上の時計の時刻が読み取れる場合は「朝」「昼」「晩」のいずれかを決定してください。
     - 朝: 4:00～11:59まで
     - 昼: 12:00～17:59まで
     - 晩: 18:00～3:59まで
  - 時刻が読み取れない場合、かつ、プロンプトから指定されていない場合は、ユーザーに対して確認してください。
- **料理のエナジー**: 画面の上部にオレンジもしくは赤い色の数字で記述される。カンマ区切りなので、カンマは除去すること
- **料理の名前**: 画面の中央部に黄色の囲みの中に記述される。
- **大成功かどうか**: 料理名の左上に「大成功」と記述してあれば大成功である。もしくは、料理のエナジーが赤色の場合は大成功である。
  - 大成功のときは、3. のスクリプトの引数に ! を指定します
  - 大成功でないときは、3. のスクリプトの引数に _ を指定します

読み取れなかった場合は、読み取れなかった項目をユーザーに伝えてください。

## 3. スクリプトの実行

スクリーンショット1枚ごとに、以下のコマンドを実行してスプレッドシートに記録してください。

```
venv\Scripts\python dish_recorder.py YYYY-MM-DD 朝|昼|晩 !|_ 料理名 エナジー
```

実行例: `venv\Scripts\python dish_recorder.py 2026-02-01 _ しんりょくアボカドグラタン 117612`

コマンドに失敗したときは出力結果とともに、そのことをユーザーに伝えてください

### 成功時の出力例

以下の内容を複数行メッセージとして送信してください。

```
📝記録しました
* 2月1日 朝 ❗244,409 (🍃しんりょくアボカドグラタン)
* 2月1日 昼 　117,612 (💤ぜったいねむりバターカレー)
```

※料理名の絵文字はスクリプトの出力に含まれます
※大成功のときは、数字の前に「❗」を付与してください
