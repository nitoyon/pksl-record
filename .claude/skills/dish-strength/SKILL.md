name: dish-strength
description: 料理エナジー読み取り
---

会話でエナジー読み取りをリクエストされたときに、次の処理をしてください。

**重要**: 処理の途中でユーザーへのテキスト出力を一切行わないでください。画像の読み取り結果、日付の決定、コマンド実行の報告などの途中経過は出力せず、すべての処理が完了した後に「出力例」の形式のみで結果を出力してください。

## 1. 日付と時間帯の決定

会話では、朝・昼・晩のいずれかと日付が指定されます。日付は曜日や昨日・今日・明日などの表現で指定されることもありますが、これらはすべてYYYY-MM-DDの形式に変換してください。

朝・昼・晩のうちの複数が指定されることもあります。その時は、添付された画像を、それぞれの時間帯に割り当てて、複数回処理を行ってください。

## 2. 添付された画像の読み取り

スクリーンショット画像から、以下の3つの値を読み取ってください。

1. **料理のエナジー**: 画面の上部にオレンジもしくは赤い色の数字で記述される。カンマ区切りなので、カンマは除去すること
2. **料理の名前**: 画面の中央部に黄色の囲みの中に記述される。
3. **大成功かどうか**: 料理名の左上に「大成功」と記述してあれば大成功である。もしくは、1. の料理のエナジーが赤色の場合は大成功である。
  - 大成功のときは、3. のスクリプトの引数に ! を指定します
  - 大成功でないときは、3. のスクリプトの引数に _ を指定します

## 3. スクリプトの実行

以下のコマンドを実行してスプレッドシートに記録してください。

```
venv\Scripts\python dish_recorder.py YYYY-MM-DD 朝|昼|晩 !|_ 料理名 エナジー
```

実行例: `venv\Scripts\python dish_recorder.py 2026-02-01 _ しんりょくアボカドグラタン 117612`

コマンドに失敗したときは出力結果とともに、そのことをユーザーに伝えてください
読み取れなかった場合は、読み取れなかった項目をユーザーに伝えてください。

### 成功時の出力例

```
📝記録しました
* 2月1日 朝 ❗244,409 (🍃しんりょくアボカドグラタン)
* 2月1日 昼 　117,612 (💤ぜったいねむりバターカレー)
```

※料理名の絵文字はスクリプトの出力に含まれます
※大成功のときは、数字の前に「❗」を付与してください
