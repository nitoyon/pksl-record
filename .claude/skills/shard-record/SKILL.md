---
name: shard-record
description: ゆめのかけら記録
---

会話でゆめのかけらの記録をリクエストされたときの処理を定めています。
複数のスクリーンショットがある場合は、そのそれぞれについて下記の処理を実施してください。

**重要**: 処理の途中でユーザーへのテキスト出力を一切行わないでください。画像の読み取り結果、日付の決定、コマンド実行の報告などの途中経過は出力せず、すべての処理が完了した後に「出力例」の形式のみで結果を出力してください。

## 1. 日付の決定

- プロンプトで日付を指定されたときには、その日付を利用してください。
   - 日付は曜日や昨日・今日・明日などの表現で指定されることもあります。
- 未指定のときは以下の手順で決定してください。
   0. スクリーンショットの左上に日付が記載してある場合はその日付を利用する。
   1. `date` コマンドを実行して現在の日時（日付と時刻）を取得する。
   2. スクリーンショットから時刻を読み取る（例: 23:11）。
   3. 手順1で取得した当日の日付とスクリーンショットの時刻を組み合わせた日時が、現在日時よりも未来かどうかを判定する。
      - 例: 現在が 2/17 0:37 で、スクリーンショットの時刻が 23:11 の場合、2/17 23:11 は未来になる。
   4. 未来になる場合は、前日の日付を採用する（上記の例では 2/16）。未来でなければ当日の日付を採用する。
- 日付はすべてYYYY-MM-DDの形式に変換してください。

## 2. 時刻の読み取り

スクリーンショットの左上から、スキル発生の時刻を hh:mm 形式で読み取ってください。

## 3. 実行者とゆめのかけら量の読み取り

スクリーンショットの下部の白い四角で囲われた範囲から、以下の情報を読み取ってください。

- **実行者**: 「〇〇のメインスキル！」の〇〇部分がポケモン名（実行者）です。
- **ゆめのかけら量**: 「ゆめのかけら x3,657をゲット！」のように表示されます。数値部分のカンマを除去して整数にしてください（例: 3,657 → 3657）。

スキルは2回発動することがあります（白い四角が2つ表示される）。この場合、それぞれについて個別にスクリプトを実行してください。

## 4. スクリプトの実行

以下のコマンドを実行してスプレッドシートに記録してください。

```
venv\Scripts\python shard_recorder.py YYYY-MM-DD hh:mm ポケモン名 かけら数
```

例: `venv\Scripts\python shard_recorder.py 2026-03-16 14:30 おてぼんね 3657`

コマンドに失敗したときは出力結果とともに、そのことをユーザーに伝えてください。
読み取れなかった場合は、読み取れなかった項目をユーザーに伝えてください。

### 出力例

以下の内容を複数行メッセージとして送信してください。

```
📝記録しました
* 3月16日 14:30 おてぼんね ✨3,657
* 3月16日 14:30 おてぼんね ✨2,104
```

※かけら数はカンマ区切りで表示してください。
