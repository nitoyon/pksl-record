---
name: daily-strength
description: エナジー読み取り
---

会話でエナジー読み取りをリクエストされたときに、添付されたPokemon Sleep のスクリーンショット画像から、以下の2つの値を読み取ってください。

1. **日付**: 画面上部に「2026年2月1日 日曜日」のように表示されている日付。YYYY-MM-DD 形式に変換する。
2. **カビゴンのエナジー**: 「カビゴンのエナジー x 睡眠スコア x ボーナス」の行に表示されている数値（カンマ区切り）。カンマを除去して整数にする。
   - 「 x ボーナス」は存在しない場合がある
   - 「ねむけパワー」の値（数億規模の大きな数値）ではないので注意。

読み取れなかった場合は、読み取れなかった項目をユーザーに伝えてください。

読み取れたら、以下のコマンドを実行してスプレッドシートに記録してください。

```
venv\Scripts\python strength_recorder.py YYYY-MM-DD エナジー値
```

実行例: `venv\Scripts\python strength_recorder.py 2026-02-01 1999819`

出力例
------

以下の内容を複数行メッセージとして送信してください。

```
📝記録しました
* 2026年1月31日のエナジー: 1,918,044
* 2026年2月1日のエナジー: 1,999,819
```

コマンドに失敗したときは出力結果とともに、そのことをユーザーに伝えてください
